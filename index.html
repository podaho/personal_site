MakTrak: Macro Nutrient Intake Manager
close

MakTrak is a web application that aims to make tracking one's macro nutritional intake easier by storing information about foods and recipes the user inputs. It provides the user useful information about their diet without the use of traditional macro tracking methods such as excel spreadsheets or calculators. My role in the project is developing back-end database structures, URL controllers, and user data processing. Using the Spring Framework, we developed this project using the MVC design pattern with HTML fragments and templates for views that change properties based on different modes of interaction. Using HQL, we access database information and map it to Java classes to persist entity relationship and information. As our team leader, I also engaged in task delegation and deadline monitoring which includes planning our project road map and informing instructors of our day to day progress and performance.

Elementary School Library Database and Form Design Software Engineering Project
more_vert
Elementary School Library Database and Form Design Software Engineering Project
close

Using Visual Studio, I've developed a functioning C# form that links to a elementary school library database in MySQL and MSSQL. The form includes features that sorts and displays database information using SQL queries through C# scripts. The project also incorporates basic encryption and decryption methods for avoiding SQL query injections through the application, as well as login, checkout, and inventory features that manipulate database information.

Thesis Abstract

Multi-agent systems (MAS) have been used to simulate interpersonal relationships and communication in a variety of studies. Using MAS, we propose the NCRAE (Neighborhood Cumulative Reward Average Evaluation) method to model interactions between individuals in a social network and interpret changes in their relationships over time. By quantifying the effects of call and SMS interactions in a given dataset among our simulation environment, we identify influences that may cause shifts in relationship closeness. Our results show a successful differentiation of close and distant relationships through their respective communication patterns and characteristics. We find that close relationships maintain a high level of interaction frequency and duration with no significant increase in closeness, whereas more distant relationships engage in little to no communication but engage in proximity interaction that defines its changes in relationship closeness.
